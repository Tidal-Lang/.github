## Code Examples

Normal Syntax:
```
var x = 5;
var y = 3.14;
novar z = x + y;
print(z);

for (var i = 0; i < 5; i = i + 1) {
    print(i);
}
```

Brainrot Syntax:
```
rizzler x = 5 no cap
rizzler y = 3.14 no cap
sigma z = x + y no cap
skibidi(z) no cap

bussin (rizzler i = 0 no cap i < 5 no cap i = i + 1 no cap) {
    skibidi(i) no cap
}
```
